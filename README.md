# openlegalexchange.org

Static website for the Open Legal Exchange (OLE) protocol.

**Live site:** [openlegalexchange.org](https://openlegalexchange.org)

## Pages

- `/` — Marketing homepage
- `/spec/` — Spec landing (links to GitHub spec repo)
- `/clear/` — CLEAR module (Legal Referral Exchange)

## Tech

Plain HTML + CSS — no framework, no build step. Drop on any static host (GitHub Pages, Netlify, Cloudflare Pages, etc.).
